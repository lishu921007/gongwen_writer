# 通知（印发类）入库判定规则 v0.1

- 适用文种：通知（印发办法/规划/方案/细则等）
- 来源依据：
  - 《语料入库规范》
  - `gw-tongzhi-20201207-001`
  - `gw-tongzhi-20220517-001`

## 放行要点
1. 标题通常包含“关于印发……的通知”。
2. 正文前部应出现主送对象（如“各省、自治区、直辖市人民政府……”）。
3. 应识别附件/被印发文件名称，例如《公共企事业单位信息公开规定制定办法》《“十四五”现代物流发展规划》。
4. 发文机关、日期、文号至少三者应具备两项以上；正式入库建议三项齐全。
5. 若页面明确标注“本文有删减”“摘要”“解读”，则不得标记为 `active`，至少降为 `limited`。

## 风险提示
- 不能把“解读稿”“答记者问”误当正式通知正文。
- 不能只摘附件正文而丢失通知抬头、文号、发文机关。
- 规划类长文若存在删节，应保留版本说明，不可伪装成全文。
