# excerpts/ 优质片段层说明

本目录保存从正式公开原文中提炼出的**优质表达片段与反例片段**，用于局部参考，而不是整篇套用。

## 子目录
- `positive/`：正例片段
- `negative/`：反例片段
- `by-doc-type/`：按文种进一步归档的片段集合

## 入库规则
- 每个片段必须绑定 `derived_from` / `parent_doc_id`。
- 必须标明：适用文种、适用章节、正反例属性、是否允许改写。
- 片段应保持语义完整，禁止只摘“漂亮句子”而脱离上下文。
- 对强时效、强地域、强事项绑定的表达，应谨慎入库或标明限用。

## 推荐命名
`{doc-type}-{section}-{slug}.md`

例如：
- `notice-opening-policy-rollout.md`
- `report-next-step-duty-compact.md`
