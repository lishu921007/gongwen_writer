# 网页聊天界面

## 功能

- 输入自然语言需求或任务卡 JSON
- 自然语言默认走 `/api/agent-run`，直接调用真实 OpenClaw agent
- 本地 fallback 链路保留在 `/api/agent-run-fallback`
- JSON 任务卡可直连 `/api/render`
- 展示处理模式、Agent、模板信息、缺失字段与正文结果
- 优化了卡片式布局、复制按钮、状态提示与移动端适配
- 新增常用提示词、快捷示例、最近输入与本地历史记录
- 新增方向 B 工作流界面：初稿 / 修订稿 / 审校意见分区
- 支持一键继续修改：更正式一点 / 更简洁一点 / 更像机关稿
- 支持一键生成审校意见
- 支持自由修订指令输入
- 支持在“初稿 / 修订稿”之间切换修订基底
- 支持按审校意见继续修订
- 第二阶段已新增六部展开区，可查看各部统一结构化意见
- 尚书省定稿已显式吸收六部的判断、发现、建议与风险/缺口

## 启动方式

### 1. 本地启动一体化 Web 服务

```bash
cd /root/.openclaw/workspace/projects/gongwen-assistant
python3 tools/run_web_app.py --host 0.0.0.0 --port 8788
```

启动后访问：

- `http://127.0.0.1:8788`
- 局域网/公网部署时可访问 `http://<服务器IP>:8788`

### 2. 公网部署建议

最简做法：

- 云服务器安全组放行 `8788`
- 或者用 Nginx/Caddy 反代到 `127.0.0.1:8788`
- 域名解析后即可公网访问

## API

- `GET /healthz`
- `POST /api/render`
- `GET /` 返回网页
